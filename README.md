# Fathy Gendy Portfolio

An interactive, digital journal and portfolio showcasing professional experience and project highlights. Built with a focus on immersive user experience and clean, modern design.

## Features

- **Interactive Experience:** Immersive page-turning mechanics for a journal-like feel.
- **Responsive Design:** Optimized for both desktop and mobile viewing with adaptive layouts.
- **Project Showcase:** Detailed view of honors, hackathons, and certifications with lightbox image support for closer inspection.
- **Modern Tech Stack:** Developed using cutting-edge web technologies for performance and maintainability.

## Tech Stack

- **Framework:** [Angular](https://angular.dev/) (Zoneless)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Rendering:** Server-Side Rendering (SSR)

## Getting Started

### Prerequisites

- Node.js (v20 or higher recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Fetoooo/fathy-gendy-portfolio.git
   cd fathy-gendy-portfolio
1. Install dependencies:
   ```bash
   npm install
2. Run the development server:
   ```bash
   npm run start
3. Then open your browser
